##  About

EnchantMenu is a feature-rich Minecraft plugin that provides an intuitive GUI-based enchantment system. Players can enchant their items through an interactive menu with dynamic pricing and intelligent item validation.

## ‚ú® Key Features

- **Interactive GUI System** - Custom inventory menu with enchantment categories
- **Economy Integration** - Vault API integration for seamless transactions
- **Citizens NPC Support** - Right-click NPCs to open enchantment menu
- **Smart Item Validation** - Automatic filtering of applicable enchantments per item type
- **Multi-Level Enchantments** - Support for various enchantment levels with scaled pricing
- **Danish Localization** - Fully localized for Danish-speaking players
- **Extensible Architecture** - Easy to add new enchantments and categories
